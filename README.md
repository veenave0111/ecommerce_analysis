# E-commerce Sales Analysis (SQL Project)

This project analyzes sales data for a small e-commerce store using SQL Server.  
It includes database tables, sample data, and SQL queries to find useful business insights.

## Files Included
- schema.sql — database and table creation
- data.sql — sample data inserts
- analysis_queries.sql — SQL queries for analysis

## Skills Used
- SQL Server
- Joins
- Group By
- Aggregations
- Basic data analysis

## Key Insights
- Total revenue generated
- Top customers by spending
- Most sold products
- Revenue by product category
- Monthly sales trend

## How to Run
1. Open SQL Server Management Studio (SSMS)
2. Run `schema.sql` to create the database and tables
3. Run `data.sql` to insert sample data
4. Run `analysis_queries.sql` to view insights
